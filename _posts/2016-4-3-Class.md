
##Class

1. Class 类的使用
    * 在面向对象的世界里，万事万物皆对象。
    * 类是对象，类是java.lang.Class类的实例对象。
    * There is a class named Class
2. 动态加载类
    * 静态加载类：编译时加载的类（用new关键字创建的类）
    * 动态加载类：运行时加载的类（用类类型创建的类，需要有接口，活抽象父类）
    * 如 Class c = Class.forName("类全名");就是动态加载。
    * 接口的引用 = c.newInstance();//创建实例对象，需要有无参构造方法。
